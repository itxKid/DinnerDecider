# Dinner Decider

Dinner Decider is an interactive React application that randomly presents food options and uses user selections to progressively eliminate choices until a final winner is selected.

### How to run
If you dont have NPM then: 
  ```sh
    npm install npm@latest -g
  ```
1. Clone the repo
```sh
  git clone https://github.com/itxKid/DinnerDecider.git
```
2. Install NPM packages
```sh
  npm install
```
3. Run Dinner Decider
```sh
  npm run dev
```
